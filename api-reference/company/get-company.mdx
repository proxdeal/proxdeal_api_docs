---
title: "Get Company"
description: "Retrieve a full company profile by its ProxDeal company ID."
openapi: "GET /v0/company/{company_id}"
---

Returns a comprehensive company profile — registration data, geocoded address, website URLs, an AI-generated description, WZ2025 industry codes, and registered representatives — aggregated from the commercial register and ProxDeal's enrichment pipeline.

Obtain a `company_id` from the [Lookup](/api-reference/lookup/lookup-by-name) or [DataFinder](/api-reference/datafinder/search-companies) endpoints. Expand the **Response** schema below for the full field reference.

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `COMPANY_NOT_FOUND` | No company with this ID exists |
| 400 | `INVALID_COMPANY_ID` | The ID contains invalid characters |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit — check `Retry-After` |
