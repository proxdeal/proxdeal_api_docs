---
title: "Get Company"
description: "Retrieve a full company profile by its ProxDeal company ID."
---

<Note>**Pricing:** See [Pricing](/pricing) for current rates.</Note>

```
GET https://api.proxdeal.com/v0/company/{company_id}
```

Returns a comprehensive company profile aggregated from multiple sources: core registration data, geocoded address, website URLs, AI-generated descriptions, WZ2025 industry classifications, and registered representatives.

## Path parameters

<ParamField path="company_id" type="string" required>
  The unique ProxDeal company identifier (e.g. `de-0001z7cs0ej81r1w`). Obtain one from the [Lookup](/api-reference/lookup/lookup-by-name) or [DataFinder](/api-reference/datafinder/search-companies) endpoints.
</ParamField>

## Example

<CodeGroup>
```bash cURL
curl https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w \
  -H "Authorization: Bearer YOUR_API_KEY"
```
```python Python
import httpx

resp = httpx.get(
    "https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
company = resp.json()["data"]
```
```typescript TypeScript
const resp = await fetch(
  "https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w",
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const { data } = await resp.json();
```
</CodeGroup>

## Response fields

<ResponseField name="status" type="string">
  Always `"success"` on a 200 response.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Company data">
    **Core registration**

    <ResponseField name="company_id" type="string">
      Unique ProxDeal company identifier.
    </ResponseField>
    <ResponseField name="company_legal_name" type="string">
      Official registered legal name.
    </ResponseField>
    <ResponseField name="company_legal_form" type="string | null">
      Legal form (e.g. `GmbH`, `AG`, `e.K.`, `Ges.m.b.H.`).
    </ResponseField>
    <ResponseField name="register_number" type="string | null">
      Commercial register number.
    </ResponseField>

      Responsible district court (Amtsgericht).
    </ResponseField>
    <ResponseField name="company_active" type="boolean | null">
      Whether the company is currently active.
    </ResponseField>
    <ResponseField name="register_type" type="string | null">
      Register type: `HRB`, `HRA`, or `FN` (Austria).
    </ResponseField>

    </ResponseField>

      Founding date in ISO 8601 format (e.g. `"2015-03-12"`).

    <ResponseField name="company_age" type="integer | null">
      Company age in full years.

    <ResponseField name="base_url" type="string | null">

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `COMPANY_NOT_FOUND` | No company with this ID exists |
| 400 | `INVALID_COMPANY_ID` | The ID contains invalid characters |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit — check `Retry-After` |
