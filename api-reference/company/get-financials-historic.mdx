---
title: "Get Historic Financials"
description: "Retrieve all available annual financial reports for a company, sorted newest first."
openapi: "GET /v0/company/{company_id}/financials/historic"
---

Returns the full time-series of all available financial reports for a company, sorted by fiscal year descending. Unlike [Get Financials](/api-reference/company/get-financials) — which returns only the most recent report — this endpoint returns every available filing from Unternehmensregister or Firmenbuch. Each report carries the same fields as the single-report endpoint.

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `FINANCIALS_NOT_FOUND` | No financial data exists for this company ID |
| 400 | `INVALID_COMPANY_ID` | The ID contains invalid characters |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
