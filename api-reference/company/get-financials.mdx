---
title: "Get Financials"
description: "Retrieve the most recent annual financial report for a company."
---

<Note>**Pricing:** €0.10 per successful call</Note>

```
GET https://api.proxdeal.com/v0/company/{company_id}/financials
```

Returns the single most recent financial report for a company, sourced from official filings — the German Unternehmensregister (Bundesanzeiger) and the Austrian Firmenbuch. Covers 25+ financial metrics including revenue, profit, equity, debt, and employee count.

For the full time-series of all available reports, use [Get Historic Financials](/api-reference/company/get-financials-historic).

## Path parameters

<ParamField path="company_id" type="string" required>
  The unique ProxDeal company identifier. Obtain one from the [Lookup](/api-reference/lookup/lookup-by-name) endpoint.
</ParamField>

## Example

<CodeGroup>
```bash cURL
curl https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w/financials \
  -H "Authorization: Bearer YOUR_API_KEY"
```
```python Python
import httpx

resp = httpx.get(
    "https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w/financials",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
fin = resp.json()["data"]
print(f"Revenue: €{fin['revenue']:,.0f}  Employees: {fin['employees']}")
```
```typescript TypeScript
const resp = await fetch(
  "https://api.proxdeal.com/v0/company/de-0001z7cs0ej81r1w/financials",
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const { data } = await resp.json();
```
</CodeGroup>

## Response fields

<ResponseField name="data" type="object">
  <Expandable title="Financials data">
    **Report metadata**

    <ResponseField name="company_id" type="string">ProxDeal company identifier.</ResponseField>
    <ResponseField name="finreport_doc_title" type="string | null">Title of the financial report document.</ResponseField>
    <ResponseField name="finreport_doc_type" type="string | null">Report type (e.g. `Jahresabschluss`, `Konzernabschluss`).</ResponseField>
    <ResponseField name="finreport_doc_id" type="string | null">Unique document identifier in the source register.</ResponseField>
    <ResponseField name="finreport_doc_source" type="string | null">Source register (e.g. `Unternehmensregister`, `Firmenbuch`).</ResponseField>
    <ResponseField name="finreport_year" type="integer | null">Fiscal year.</ResponseField>
    <ResponseField name="finreport_date_publish" type="string | null">Date the report was published (ISO 8601).</ResponseField>

    <ResponseField name="total_assets" type="number | null">Total assets (Bilanzsumme).</ResponseField>
    <ResponseField name="equity" type="number | null">Equity (Eigenkapital).</ResponseField>
    <ResponseField name="debt" type="number | null">Debt / liabilities (Verbindlichkeiten).</ResponseField>
    <ResponseField name="liabilities" type="number | null">Total liabilities.</ResponseField>
    <ResponseField name="cash_balance" type="number | null">Cash and cash equivalents.</ResponseField>
    <ResponseField name="current_assets" type="number | null">Current assets (Umlaufvermögen).</ResponseField>
    <ResponseField name="fixed_assets" type="number | null">Fixed assets (Anlagevermögen).</ResponseField>
    <ResponseField name="receivables" type="number | null">Receivables (Forderungen).</ResponseField>
    <ResponseField name="provisions" type="number | null">Provisions (Rückstellungen).</ResponseField>

    **Ratios**

    <ResponseField name="equity_ratio" type="number | null">Equity ratio — `equity / total_assets` (0.0–1.0).</ResponseField>
    <ResponseField name="debt_ratio" type="number | null">Debt ratio — `debt / total_assets` (0.0–1.0).</ResponseField>
    <ResponseField name="debt_equity_ratio" type="number | null">Debt-to-equity ratio (Verschuldungsgrad).</ResponseField>

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `FINANCIALS_NOT_FOUND` | No financial data exists for this company ID |
| 400 | `INVALID_COMPANY_ID` | The ID contains invalid characters |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
