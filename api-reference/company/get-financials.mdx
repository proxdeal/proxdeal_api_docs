---
title: "Get Financials"
description: "Retrieve the most recent annual financial report for a company."
openapi: "GET /v0/company/{company_id}/financials"
---

Returns the single most recent financial report for a company, sourced from official filings — the German Unternehmensregister (Bundesanzeiger) and the Austrian Firmenbuch. Covers 25+ financial metrics including revenue, profit, equity, debt, and employee count, all in EUR.

Obtain a `company_id` from the [Lookup](/api-reference/lookup/lookup-by-name) endpoint. For the full time-series of all available reports, use [Get Historic Financials](/api-reference/company/get-financials-historic).

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `FINANCIALS_NOT_FOUND` | No financial data exists for this company ID |
| 400 | `INVALID_COMPANY_ID` | The ID contains invalid characters |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
