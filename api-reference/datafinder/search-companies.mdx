Filter-based company discovery across the full ProxDeal DACH database. Supports six filter groups: legal attributes, geography, financial metrics, industry, web presence, and description keywords. Filters across groups are combined with AND logic, and the endpoint returns a paginated list of matching companies.

Unlike the [Lookup](/api-reference/lookup/lookup-by-name) endpoints (which resolve a specific company), the DataFinder is designed for structured discovery â€” for example, finding all active GmbHs in Bayern with revenue above â‚¬1M. At least one field across any group must be set.

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `NO_FILTERS` | No filter fields were set |
| 400 | `INVALID_PARAMETER` | Input contains null bytes |
| 400 | `LOC_INPUT_CONFLICT` | `loc_input` used with explicit `loc_lat`/`loc_lng` |
| 400 | `LOC_INPUT_MISSING_RADIUS` | `loc_input` set but `loc_radius` missing |
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 422 | `GEOCODE_FAILED` | `loc_input` could not be resolved to coordinates |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
