---
title: "Get Dimensions"
description: "Retrieve the 12 business model dimensions for a company by website URL."
---

<Note>**Pricing:** See [Pricing](/pricing) for current rates.</Note>

```
GET https://api.proxdeal.com/v0/dimensions?url={url}
```

Returns the 12 business model dimension classifications for any URL that has already been scraped and classified. The URL is normalised to its bare domain before matching.

If no classification exists for the URL yet, use [Analyze Website](/api-reference/website/analyze-website) to trigger scraping and classification first.

## Query parameters

<ParamField query="url" type="string" required>
  Website URL. Normalised to bare domain before matching (protocol, `www.`, paths, and query parameters are stripped).
</ParamField>

## Example

<CodeGroup>
```bash cURL
curl "https://api.proxdeal.com/v0/dimensions?url=https://www.beispiel.de" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
```python Python
import httpx

resp = httpx.get(
    "https://api.proxdeal.com/v0/dimensions",
    params={"url": "https://www.beispiel.de"},
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
dims = resp.json()["data"]
```
</CodeGroup>

## Dimension reference

Each dimension uses a fixed controlled vocabulary. Single-select dimensions return a string; multi-select return a string array (primary value first).

| Dimension | Select | Values |
|---|---|---|
| `d1_value_delivery` | multi | `Digital_Software`, `Physical_Hardware`, `Human_Service`, `Asset_Rental`, `DeepTech_RnD_IP`, `Aggregation_Resale`, `Capital_Portfolio` |
| `d2_value_chain_position` | single | `Platform_Infrastructure`, `Enabling_Middleware`, `Application_EndUser`, `Aggregator_Orchestrator`, `LastMile_Fulfillment` |
| `d3_customer_type` | multi | `B2G`, `B2B_Enterprise`, `B2B_SME`, `B2C`, `B2B2C`, `B2D_Developer` |
| `d4_market_positioning` | single | `Category_Creator`, `Disruptor_Challenger`, `Fast_Follower`, `Established_Leader`, `Niche_Specialist`, `Local_Regional`, `Commodity_Competitor` |
| `d5_revenue_model` | multi | `Subscription_Recurring`, `Usage_Metered`, `Transactional_OneSale`, `Project_SOW`, `Rental_Lease`, `Licensing_Royalties`, `Commission_TakeRate`, `Advertising`, `ManagementFee_AUM` |
| `d6_pricing_mechanism` | multi (1–2) | `Per_Seat_User`, `Tiered_Volume`, `Flat_Rate`, `Per_Unit_Consumption`, `Fixed_Price_Catalog`, `Value_Outcome_Linked`, `Negotiated_Custom` |
| `d7_gtm_motion` | multi | `Product_Led_SelfServe`, `Sales_Led_Direct`, `Channel_Partner_Reseller`, `Marketplace_Listing`, `Procurement_RFP`, `Local_WalkIn_Referral`, `Ecommerce_OnlineDirect` |
| `d8_expansion_mechanic` | multi (1–2) | `Seat_User_Expansion`, `Usage_Consumption_Growth`, `Module_CrossSell`, `Price_Escalation`, `Repeat_Purchase`, `Project_Scope_Expansion`, `None_Static` |
| `d9_scaling_engine` | single | `Network_Effects`, `Data_Flywheel`, `Economies_of_Scale`, `Brand_Reputation`, `Capital_Deployment`, `Linear_People_Dependent` |
| `d10_lock_in_mechanism` | multi (1–2) | `Technical_Integration`, `Data_Gravity`, `Contractual_SLA`, `Workflow_Process_Embed`, `Relationship_Trust_Dependency`, `Regulatory_Compliance`, `Low_Easily_Replaceable` |
| `d11_growth_bottleneck` | multi (1–2) | `Talent_Acquisition`, `Capital_CAPEX`, `Regulatory_Approval`, `Market_Size_TAM`, `Technology_Maturity`, `Zero_Friction_Scalable` |
| `d12_geographic_scope` | single | `Hyperlocal`, `Regional`, `National`, `Multi_Country`, `Global` |

<ResponseField name="data.company_id" type="null">
  Always `null` — no ProxDeal `company_id` is resolved from a URL lookup.
</ResponseField>

<ResponseField name="data.quantified_at" type="string">
  ISO 8601 timestamp of when the classification was last computed.
</ResponseField>

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `INVALID_URL` | No valid domain could be extracted |
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `DIMENSIONS_NOT_FOUND` | URL not yet classified — use `POST /v0/website/analyze` first |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
