<Note>**Pricing:** See [Pricing](/pricing) for current rates.</Note>

```
POST https://api.proxdeal.com/v0/website/analyze
```

Given any website URL, returns structured data extracted from the company website plus all 12 business model dimension classifications. If the website has already been scraped and classified, cached results are returned immediately. Otherwise, the website is scraped on demand and the AI classification runs before the response is returned.

The `source` field in the response indicates which path was taken.

<Warning>
  Response time varies significantly by path:
  - `"cache"` — both scraped data and dimensions already existed → **under 1 second**
  - `"fresh_quantify"` — scraped data existed, dimensions freshly generated → **10–20 seconds**
  - `"fresh_scrape"` — website scraped and classified for the first time → **40–90 seconds**

  Set your client timeout accordingly. A 120-second timeout is recommended.
</Warning>

## Request body

<ParamField body="url" type="string" required>
  Website URL to analyze. Accepts any common format — protocol, `www.`, paths, and query parameters are all stripped before matching.
</ParamField>

## Example

<CodeGroup>
```bash cURL
curl -X POST https://api.proxdeal.com/v0/website/analyze \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.beispiel.de"}'
```
```python Python
import httpx

resp = httpx.post(
    "https://api.proxdeal.com/v0/website/analyze",
    json={"url": "https://www.beispiel.de"},
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    timeout=120.0,
)
result = resp.json()["data"]
print(result["source"])          # "cache" | "fresh_quantify" | "fresh_scrape"
print(result["scraped"]["offering_description"])
print(result["dimensions"]["d5_revenue_model"])
```
```typescript TypeScript
const resp = await fetch("https://api.proxdeal.com/v0/website/analyze", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({ url: "https://www.beispiel.de" }),
  signal: AbortSignal.timeout(120_000),
});
const { data } = await resp.json();
```
</CodeGroup>

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 422 | `INVALID_URL` | No valid domain could be extracted from the input |
| 422 | `SCRAPE_FAILED` | Network timeout, TLS failure, or similar scraper error |
| 422 | `SCRAPE_INSUFFICIENT` | Website reached but did not yield enough content — often bot-blocked or login-gated |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
