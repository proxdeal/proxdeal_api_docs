---
title: "Analyze Website"
description: "Scrape and classify any company website on demand."
openapi: "POST /v0/website/analyze"
---

Given any website URL, returns structured data extracted from the company website plus all 12 business model dimension classifications. If the website has already been scraped and classified, cached results are returned immediately. Otherwise, the website is scraped on demand and the AI classification runs before the response is returned.

The `source` field in the response indicates which path was taken. For dimensions only on an already-classified URL, use [Get Dimensions](/api-reference/website/get-dimensions).

<Warning>
  Response time varies significantly by path:
  - `"cache"` — both scraped data and dimensions already existed → **under 1 second**
  - `"fresh_quantify"` — scraped data existed, dimensions freshly generated → **10–20 seconds**
  - `"fresh_scrape"` — website scraped and classified for the first time → **40–90 seconds**

  Set your client timeout accordingly. A 120-second timeout is recommended.
</Warning>

## Errors

| Status | Code | Cause |
|---|---|---|
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 422 | `INVALID_URL` | No valid domain could be extracted from the input |
| 422 | `SCRAPE_FAILED` | Network timeout, TLS failure, or similar scraper error |
| 422 | `SCRAPE_INSUFFICIENT` | Website reached but did not yield enough content — often bot-blocked or login-gated |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
