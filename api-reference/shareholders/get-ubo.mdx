---
title: "Ultimate Beneficial Owners (UBO)"
description: "Recursively identify the natural persons at the top of an ownership chain."
openapi: "GET /v0/company/{company_id}/shareholders/ubo"
---

Recursively traverses the ownership chain to identify ultimate beneficial owners — the natural persons (or unresolvable entities) at the top of the ownership structure. Traversal is capped at **5 levels** deep; if it is cut short there, `max_depth_reached` is `true`.

If a corporate shareholder cannot be found in the ProxDeal database, it is returned as a terminal node with `chain_resolved: false`. Each result's `ownership_chain` lists the `company_id` path from the queried company to that owner.

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `INVALID_COMPANY_ID` | ID contains invalid characters |
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `SHAREHOLDERS_NOT_FOUND` | No shareholder data exists for this company ID |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
