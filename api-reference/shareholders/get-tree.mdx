---
title: "Shareholder Ownership Tree"
description: "Get the full nested ownership tree rooted at a company."
openapi: "GET /v0/company/{company_id}/shareholders/tree"
---

Returns the full nested ownership tree rooted at the queried company. Corporate shareholder nodes expand recursively up to **5 levels** deep; natural persons and unresolvable companies are leaf nodes. Each node's `shareholders` array follows the same recursive structure.

Use this endpoint when you need the complete ownership picture in a single hierarchical structure — for due diligence, compliance screening, or visualising corporate trees. For a flat list of the ultimate owners instead, use [Get UBOs](/api-reference/shareholders/get-ubo).

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `INVALID_COMPANY_ID` | ID contains invalid characters |
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `SHAREHOLDERS_NOT_FOUND` | No shareholder data exists for this company ID |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
