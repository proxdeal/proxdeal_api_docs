---
title: "Direct Shareholders (L1)"
description: "Get all direct (Level 1) shareholders of a company."
---

<Note>**Pricing:** See [Pricing](/pricing) for current rates.</Note>

```
GET https://api.proxdeal.com/v0/company/{company_id}/shareholders
```

Returns all direct shareholders of a company as recorded in the commercial register (Unternehmensregister / Firmenbuch). Includes both natural persons and corporate entities, with ownership percentages and capital contributions where available.

For the full ownership chain up to the ultimate natural persons, use [Get UBOs](/api-reference/shareholders/get-ubo). For a nested tree view, use [Get Shareholder Tree](/api-reference/shareholders/get-tree).

## Path parameters

<ParamField path="company_id" type="string" required>
  The unique ProxDeal company identifier.
</ParamField>

## Response fields

<ResponseField name="data" type="object[]">
  List of direct shareholders.

  <Expandable title="Shareholder item">
    <ResponseField name="shareholder_type" type="string | null">`"person"`, `"company"`, or `"other"`.</ResponseField>
    <ResponseField name="is_natural_person" type="boolean | null">Whether the shareholder is a natural person.</ResponseField>
    <ResponseField name="ownership_form" type="string | null">Ownership form (e.g. `"direct"`, `"indirect"`).</ResponseField>
    <ResponseField name="percent" type="number | null">Ownership percentage (0–100).</ResponseField>
    <ResponseField name="kapitaleinlage" type="number | null">Capital contribution in EUR.</ResponseField>
    <ResponseField name="currency" type="string | null">Currency of the capital contribution.</ResponseField>

    **Natural person fields** (populated when `is_natural_person=true`)

    <ResponseField name="shareholder_person_first_name" type="string | null">First name.</ResponseField>
    <ResponseField name="shareholder_person_last_name" type="string | null">Last name.</ResponseField>
    <ResponseField name="shareholder_person_birth_date" type="string | null">Date of birth (ISO 8601).</ResponseField>

    **Company fields** (populated when `is_natural_person=false`)

    <ResponseField name="shareholder_company_legal_name" type="string | null">Legal name of the shareholder company.</ResponseField>
    <ResponseField name="shareholder_company_legal_form" type="string | null">Legal form (e.g. `GmbH`, `AG`).</ResponseField>
    <ResponseField name="shareholder_company_register_number" type="string | null">Commercial register number.</ResponseField>
    <ResponseField name="shareholder_company_district_court" type="string | null">Responsible district court.</ResponseField>
    <ResponseField name="shareholder_company_register_type" type="string | null">Register type.</ResponseField>
    <ResponseField name="shareholder_company_founding_date" type="string | null">Founding date of the shareholder company.</ResponseField>
    <ResponseField name="shareholder_company_governing_law" type="string | null">Governing law / jurisdiction.</ResponseField>
    <ResponseField name="shareholder_company_incorp_country" type="string | null">Country of incorporation.</ResponseField>

    **Address**

    <ResponseField name="shareholder_seat_address_city" type="string | null">City of the registered seat.</ResponseField>
    <ResponseField name="shareholder_seat_address_state" type="string | null">State / Bundesland.</ResponseField>
    <ResponseField name="shareholder_seat_address_country" type="string | null">Country.</ResponseField>

    **Cross-reference**

    <ResponseField name="shareholder_sdac_id" type="string | null">
      ProxDeal `company_id` of the shareholder company, if it could be resolved in the database. Use this to fetch the shareholder's own profile or ownership structure.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="count" type="integer">Number of shareholders returned.</ResponseField>

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `INVALID_COMPANY_ID` | ID contains invalid characters |
| 401 | `AUTH_INVALID_KEY` | Invalid or missing API key |
| 404 | `SHAREHOLDERS_NOT_FOUND` | No shareholder data exists for this company ID |
| 429 | `RATE_LIMIT_EXCEEDED` | Rate limit hit |
