---
title: "Identify Company"
description: "Resolve free text to a ProxDeal company ID using an LLM pipeline."
openapi: "POST /v0/identify/company"
---

Resolves a free-text string to a `company_id` in the ProxDeal DACH database using a two-stage LLM pipeline: field extraction followed by candidate ranking. Returns the top match with a confidence score plus up to 4 alternative candidates. The richer the input — name, location, register number — the better the match quality.

**You are only charged when a company is successfully resolved.** If the pipeline cannot match the input, no credit is deducted. For bulk resolution of a CSV or Excel file, use [Identify Batch](/api-reference/identify/identify-batch).

<Warning>
  This endpoint calls an LLM pipeline. Typical response time is 3–10 seconds. Set your client timeout to at least 60 seconds.
</Warning>
