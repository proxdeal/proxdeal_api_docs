title: "Pricing"
description: "Credit costs for every ProxDeal API endpoint."
---

Credits are deducted only on **successful** responses (HTTP 2xx). Failed calls — auth errors, validation errors, 5xx — are never charged.

The live rates below are fetched directly from the API and are always up to date.

```
GET https://api.proxdeal.com/v0/pricing
```

No authentication required.

## Fetching rates programmatically

<CodeGroup>
```bash cURL
curl https://api.proxdeal.com/v0/pricing
```
```python Python
import httpx

resp = httpx.get("https://api.proxdeal.com/v0/pricing")
rates = resp.json()["data"]
# rates["get_company"] → {"eur_cents": 10, "eur": 0.10}
```
```typescript TypeScript
const resp = await fetch("https://api.proxdeal.com/v0/pricing");
const { data } = await resp.json();
// data.get_company → { eur_cents: 10, eur: 0.10 }
```
</CodeGroup>
