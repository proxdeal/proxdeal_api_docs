The ProxDeal API gives you structured access to comprehensive company data across Germany, Austria, and Switzerland. It is built on top of official commercial register sources (Unternehmensregister, Firmenbuch, Handelsregister) enriched with AI-generated descriptions, website data, financial reports, and ownership structures.

**Base URL:** `https://api.proxdeal.com`

**Current version:** v0.1.0

| Austria | Firmenbuch | ~600 K companies |
| Switzerland | Handelsregister | ~700 K companies |

## Getting started

<Steps>
  <Step title="Get an API key">
    Sign up at [app.proxdeal.com](https://app.proxdeal.com) and create a project. Generate an API key from the **API Settings** tab.
  </Step>
  <Step title="Make your first request">
    ```bash
    curl https://api.proxdeal.com/v0/lookup/company?query=Siemens \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Step>
  <Step title="Fetch the full profile">
    Use the `company_id` from the lookup response to pull the full company record.
    ```bash
    curl https://api.proxdeal.com/v0/company/de-COMPANY_ID \
      -H "Authorization: Bearer YOUR_API_KEY"
    ```
  </Step>
</Steps>
